# AI Engineering Path (.NET)

Репозиторий-журнал для перехода в сферу AI Engineering и AI Quality Assurance. 
Проект сфокусирован на экосистеме Microsoft, C# и современных паттернах интеграции LLM.

## Структура репозитория

- `/docs` — архитектурные схемы, конспекты теории и глоссарий.
- `/src` — исходный код проектов и автономных агентов.
- `/tests` — инфраструктура для оценки качества генерации (LLM-as-a-judge, Red Teaming).

## Главный проект: AI Code Reviewer
Первым практическим шагом является разработка собственного ИИ-агента для автоматизированного код-ревью C# проектов с использованием `Microsoft.Extensions.AI`.
