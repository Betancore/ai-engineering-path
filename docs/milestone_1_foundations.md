# Milestone 1: Foundations & AI Code Review Agent

**Цель:** Изучить базовую работу с API моделей и написать полезный C# инструмент для инспекции кода.

## Архитектура AiCodeReviewer (v1.0)
Утилита должна принимать на вход путь к папке `src`, собирать содержимое всех `.cs` файлов в единый промпт и отправлять его в LLM с инструкцией провести аудит архитектуры (SOLID, нейминг, уязвимости).

## План действий

### 1. Теория: Понимание инструмента
- [ ] Токены: как модель "видит" текст. Контекстное окно.
- [ ] Роли в API: System, User, Assistant. 
- [ ] Параметры детерминированности: `Temperature` и `Top-P` (для код-ревью нужна нулевая температура).

### 2. Настройка среды
- [ ] Получить API-ключ (Gemini API / OpenAI).
- [ ] Настроить хранение ключа через `dotnet user-secrets`.
- [ ] Создать проект `AiCodeReviewer` (.NET 8/9).
- [ ] Подключить NuGet `Microsoft.Extensions.AI`.

### 3. Разработка логики
- [ ] Написать `LocalFileScanner` для сбора текста из `.cs` файлов.
- [ ] Написать системный промпт для роли Senior C# Engineer.
- [ ] Сделать вызов к модели и получить ответ.
- [ ] *Продвинутый шаг:* Заставить модель возвращать не просто текст, а строго типизированный JSON с массивом найденных проблем (Structured Output).
