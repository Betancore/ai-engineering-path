# Global AI Engineering Roadmap (C# / .NET)

## Milestones

### Milestone 1: Foundations & The Code Reviewer (Current)
* **Теория:** Механика LLM, токены, параметры генерации, системные промпты.
* **Практика:** Создание утилиты `AiCodeReviewer`. Сборка контекста из `.cs` файлов, отправка в LLM через `Microsoft.Extensions.AI`, парсинг структурированного ответа.

### Milestone 2: Context & Retrieval (RAG)
* **Теория:** Эмбеддинги, стратегии Chunking (нарезка текста), векторные базы данных.
* **Практика:** Интеграция локальной БД (Qdrant), реализация RAG-пайплайна для поиска ответов по документации.

### Milestone 3: Agents & Semantic Kernel
* **Теория:** Агенты, планировщики (Planners), паттерн ReAct.
* **Практика:** Переход на **Microsoft Semantic Kernel**. Написание нативных C#-плагинов, которые модель сможет вызывать самостоятельно.

### Milestone 4: AI QA & Evaluation (LLM-as-a-Judge)
* **Теория:** Метрики оценки RAG (Faithfulness, Relevance), фреймворк RAGAS.
* **Практика:** Написание автоматизированных тестов (xUnit/NUnit) для проверки ответов ИИ. Реализация пайплайна, где сильная модель оценивает слабую.

### Milestone 5: LLMOps & Security
* **Теория:** Трейсинг, мониторинг токенов, Prompt Injection, Jailbreaks.
* **Практика:** Настройка телеметрии для агентов, написание автоматизированных атак на свои же системы для проверки их устойчивости.
